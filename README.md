##Student_Performance_Analysis
A Python data analysis project exploring social and academic factors affecting student grades.
##Project Overview
This project uses Python to analyse a dataset of students performance ina a Portuguese language course.I explored how variables like study time, absences and social habits correlate with final grade.
##Key Insights
Absence Threshold:Discovered a significant grade drop-off for students with more than 10 absences.
Social impact:Identified a negative correlation beteween frequent weekend social activities and academic success.
Family Influence:Analyzed the impact of parental education levels on student oucomes.
##Tools used:
Python-core language
Pandas-Data Manipulation and cleaning
Seaborn & Matplotlib-Data visualization
